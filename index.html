You will need a GitHub account, please setup a public repo to upload your solutions to Problem 1, 2 & 3. Let your facilitator know your repos URL.

When writing your code, please think about SOLID principals and writing good clean code (Readable, Testable, Maintainable, Efficient, Clean)

Problem 1 (2hrs)

Go to this Skynet CodinGame and solve it so all tests pass. Once it passes, upload your solution to your GitHub repo.

Problem 2 (2hrs)

Look at Validation, and have a look at what we want:

You need to:

  • Add Password validation type to first input
  • Add a second input box with Numbers and Passwords validation types.

Passwords may contain letters, numbers and at least 1 of the following symbols (!@#&*), and it can not have 3 consecutive numbers, these will fail abc123, Bob789, because they have 3 consecutive numbers. These will pass, abc135, Bob987.

In your command line, type npm run mocha to run tests.

Problem 3 (1.5hrs)

Look at Optimisation, in Chrome DevTools press Ctrl+Shift+P, type in Performance and select "Drawer Performance Monitor". Then unselect all options and select "Layouts / sec". Get the Layouts a second down to under 60.
